# アイコン・マッチング・ゲーム プレゼンテーション

## ゲーム概要
レベルを選んで自分のペースで楽しめるアイコン・マッチング・ゲーム。ムーブ数と時間で競い、AIが解いてくれるから手軽に挑戦。リーダーボードで記録を残せ、初めてでも安心して遊べます。

## アピールポイント
- 自分のレベルに合わせて難易度を選べ、挫折せずに続けられる
- ムーブ数と時間で競えるリーダーボードで、達成感が味わえる
- AIが自動で解いてくれるので、ヒントをもらいながら楽しめる

## ゲームの特徴
- **レベル選択**: Easy, Medium, Hard から選べます。
- **統計表示**: ムーブ数、時間、スター評価が見えます。
- **コントロール**: リスタートボタン、AIソルブボタン。
- **リーダーボード**: 各レベルのトップ10を表示。
- **テーマ切り替え**: ライト/ダークモード。
- **勝利モーダル**: ゲームクリア時に統計を表示。

## 遊び方
1. レベルを選択。
2. カードをクリックしてめくり、同じアイコンをマッチさせる。
3. すべてのペアをマッチさせたら勝利。
4. 名前を入力してリーダーボードに記録。

## 技術仕様
- HTML, CSS, JavaScript で作成。
- Firebase を使用したリーダーボード。
- 音声効果付き。
- レスポンシブデザイン。
