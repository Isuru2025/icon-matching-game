# 🎮 アイコン・マッチング・ゲーム

**直感操作 × 自分のペース × AIサポート**  
誰でも気軽に挑戦できる、モダンなマッチングパズルゲームです。

---

## 🧩 ゲーム概要

このゲームは、**レベルを選んで自分のペースで楽しめる**  
アイコン・マッチング・パズルです。

- ムーブ数とクリア時間を競うシンプルなルール
- 行き詰まっても **AIが解いてくれる安心設計**
- リーダーボードで記録を残し、何度でも挑戦可能

**初めてでも、失敗しても、楽しい。**  
そんな体験を目指して作られています。

---

## ✨ アピールポイント

- 🎯 **難易度選択式**  
  Easy / Medium / Hard から選べるので、挫折せずに続けられます。

- 🏆 **リーダーボード機能**  
  ムーブ数と時間で競い、達成感と成長を実感できます。

- 🤖 **AIソルブ機能**  
  自動で解いてくれるので、ヒントとしても学習用としても使えます。

---

## 🚀 ゲームの特徴

### 🎚 レベル選択
- Easy / Medium / Hard の3段階

### 📊 統計表示
- ムーブ数
- 経過時間
- スター評価（プレイ内容に応じて変化）

### 🎮 操作性
- リスタートボタン
- AI ソルブボタン

### 🏅 リーダーボード
- 各レベルごとに **トップ10** を表示
- 名前を入力してスコアを保存

### 🌗 テーマ切り替え
- ライトモード / ダークモード対応

### 🎉 勝利モーダル
- クリア時に結果をポップアップ表示

---

## 🕹 遊び方

1. レベルを選択します  
2. カードをクリックしてめくります  
3. 同じアイコンを見つけてマッチさせます  
4. すべてのペアを揃えるとゲームクリア  
5. 名前を入力してリーダーボードに記録！

---

## 🛠 技術仕様

- **HTML / CSS / JavaScript**
- **Firebase**（リーダーボード管理）
- 効果音付き（操作・クリア時）
- **レスポンシブデザイン対応**
  - PC / タブレット / スマートフォン

---

## 🎯 このゲームが目指すもの

- 短時間でも楽しめる手軽さ
- 何度も挑戦したくなる設計
- ゲームが苦手な人でも安心できる体験
